# 💘 연애 권력 불균형 진단기

> 카카오톡 대화를 AI로 분석해 연인 사이의 우위관계를 수치화하는 Streamlit 웹앱

---

## 프로젝트 개요

"왜 항상 내가 먼저 연락하지?" "왜 나만 빨리 답장하지?"

연인 관계에는 명확하게 드러나지 않는 권련 구조가 존재합니다.
이 프로젝트는 카카오톡 PC 내보내기(`.csv`) 파일을 입력받아, 
HuggingFace 기반 NLP 모델로 대화를 분석하고 두 가지 핵심 지표를 수치화합니다.
